# Community Health Access Analytics

This portfolio project ranks Maryland counties for community-health outreach using official public datasets from CDC, HRSA, and CMS. It is designed to show practical analyst skills: data sourcing, ETL, data quality handling, feature engineering, transparent scoring, and executive-style reporting.

Live dashboard: https://bnjmnprto.github.io/community-health-access-analytics/

## Business Question

Where should a health system, nonprofit, or public health team prioritize outreach when balancing chronic-disease burden, social barriers, provider shortages, and local hospital access?

## Data Sources

- CDC PLACES 2025 county GIS-friendly data: health outcomes, access, social needs, and county population.
- HRSA Health Professional Shortage Areas: active primary-care and mental-health HPSA designations and scores.
- CMS Hospital General Information: hospital counts, emergency-service status, and overall ratings.

## Methodology

The pipeline creates a county-level priority score:

`Priority = 40% health burden + 35% social need + 25% access gap`

Each input metric is min-max normalized across Maryland counties so that higher values represent higher observed need within the state.

Health burden includes diabetes, obesity, high blood pressure, physical distress, mental distress, and depression. Social need includes uninsured adults, food insecurity, housing insecurity, transportation barriers, and utility insecurity. Access gap combines active HPSA designations, HPSA score severity, hospital count per 100,000 residents, and CMS star-rating gaps where available.

The ETL uses county FIPS codes as the join key and explicitly preserves Baltimore City (`24510`) separately from Baltimore County (`24005`), a common Maryland data-quality edge case.

## Repository Structure

- `src/build_dashboard.py`: reproducible ETL, scoring, CSV export, and dashboard generation.
- `data/processed/md_county_health_access_scores.csv`: processed county-level model output.
- `docs/index.html`: static dashboard published with GitHub Pages.
- `docs/md_county_health_access_scores.csv`: downloadable dashboard dataset.

## Reproduce

```bash
python3 src/build_dashboard.py
```

The script uses only the Python standard library. No third-party packages are required.
